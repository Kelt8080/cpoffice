Introduction:

這一個系統是採用windows chrome為主體架構，結合goole driver線上免費服務製作出來的個人雲端辦公平台。

安裝步驟:

  • 請用Chrome開啟本系統
  • 如果系統無法自動跳轉，請依照下圖操作: 一律顯示彈出式視窗
  • 安裝chrome add-on: LocalLinks 讓Chrome可以開啟本機程式
  • 安裝chrome add-on: iChrome 讓iframe支援google系列產品
  • 安裝chrome add-on: Ignore X-Frame headers 讓iframe支援https的網頁

Prominent Features:

手機UI:模擬手機操作介面，支援觸控螢幕。

自訂各頁分類主題:使用者可以依照自己的需求，任意地把本機軟體，APP與網頁程式整合在一個手機頁面中。

開源:本系統存放於github，歡迎各方黑客協同開發。

個人化:運用APP的帳號認證模式，不同的帳號進入不同的網頁。

語音控制:這是進階功能，可以將你的NB化身成你的個人AI，需搭配個人命令辭庫與對應的個人化自動功能，有興趣請聯絡 Kelt Chen
